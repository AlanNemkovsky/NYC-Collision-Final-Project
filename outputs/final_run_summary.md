# Final run summary for paper writing

This file summarizes the outputs generated by the local pipeline and used in the finalized paper and presentation.

## Key files

- `tables/model_metrics.csv`: main model comparison
- `tables/ablation_metrics.csv`: feature-source ablation results
- `tables/cluster_profiles.csv`: unsupervised cluster summaries, if clustering ran
- `reports/best_model_summary.txt`: main modeling notes
- `reports/data_quality_report.txt`: data quality notes, if available
- `figures/`: paper-ready figures
